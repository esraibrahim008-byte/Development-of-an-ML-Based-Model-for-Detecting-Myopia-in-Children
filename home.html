ML-Based Model for Detecting Myopia in Children

Welcome to the ML-Based Model for Detecting Myopia in Chsildrsen.
This application uses machine learning to predict myopia in children.
This web-based application utilizes machine learning techniques to support
the early detection of myopia in children. The system analyzes key
demographic, behavioral, and clinical factors,
including age, sports activity hours, television viewing hours, parental history of myopia,
and SPHEQ measurements to generate prediction results.
